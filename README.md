This Discontinuous Conduction Mode (DCM), non-isolated, 800-W SEPIC converter battery charger provides 
an adjustable output voltage and current. The output voltage can be adjusted from 282 V to 400 V by setting 
an external DC voltage between 0 V to 3 V. The output charging current can be adjusted between 0 A to 2 
A by adjusting the duty cycle of a PWM input. It operates over an input voltage range of 340 Vdc–420 Vdc. 
Independent voltage and current feedback loops are used to maintain regulation
